‚ö° Heart of Ender (Obsidian Pinnacle)

Health: 10,000 | Shinn Difficulty: S

The Ultimate Challenge - 3 Phases

Phase 1 (100-75% HP): Energy orbs, laser beams

  • Dodge red orbs by circling
  • Use pillars to block lasers
  • Attack during orb cooldown

Phase 2 (75-50% HP): Shadow clones, teleportation

  • Watch for teleport indicators
  • Attack real heart (has health bar)
  • Keep moving to avoid shadow strikes

Phase 3 (50-0% HP): Enrage mode - all attacks faster

  • Save all artifacts for this phase
  • Use Iron Hide during intense moments
  • Stay calm and stick to dodge patterns

Shinn's Speed Strategy:

  • Save Death Cap Mushroom for phase 3
  • Use Iron Hide during laser beam sequences
  • Learn the orb patterns - they're predictable
  • Don't panic during enrage - stay methodical

Recommended Build: Death Cap + Iron Hide + Corrupted Beacon + Soul gear

Difficulty: 9.5/10 | Speedrun Time: 6-8 minutes

üëë Arch-Illager (Final Boss)

Health: 12,000 | Shinn Difficulty: S+

The Ultimate Test - 3 Epic Phases

Phase 1 (100-66% HP): Basic magic attacks, orb shields

  • Break purple shields quickly
  • Dodge energy orbs in patterns
  • Attack during shield cooldown

Phase 2 (66-33% HP): Enhanced magic, shadow clones

  • More frequent shield regeneration
  • Faster orb attacks
  • Shadow clones appear periodically

Phase 3 (33-0% HP): True power unleashed

  • Rapid-fire magic attacks
  • Constant shadow clones
  • Enraged state - all attacks faster
  • Multiple shield layers

Shinn's Championship Strategy:

  1. Preparation: Full soul build with Wither Armor
  2. Phase 1: Conservative approach, learn patterns
  3. Phase 2: Use Death Cap Mushroom for burst
  4. Phase 3: All artifacts + potions, full commitment
  5. Survival: Save Iron Hide for emergency situations

Recommended Build (Shinn Approved):

  • Weapon: Eternal Knife (soul gathering)
  • Armor: Wither Armor (soul boosting)
  • Artifacts: Death Cap Mushroom, Corrupted Beacon, Iron Hide Amulet
  • Strategy: Soul generation ‚Üí Corrupted Beacon spam ‚Üí Iron Hide for survival
Ultimate Difficulty: 10/10 | Speedrun Time: 8-12 minutes
